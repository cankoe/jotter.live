Jotter

Your quick notepad.
Offline. Private. Yours.

Markdown · Files & Images · Works Offline · No Account

Everything stays in your browser. No servers. No tracking.

Privacy first

Jotter stores everything locally in your browser using IndexedDB and OPFS. Your notes never leave your device. No accounts, no analytics, no cookies, no servers.

Features

  • Rich markdown editor with live preview — headings, bold, italic, lists, checklists, tables
  • Syntax-highlighted code blocks with language labels and copy button
  • LaTeX math rendering (inline and block) via KaTeX
  • Mermaid diagrams — flowcharts, sequence diagrams, and more
  • Paste, drag-drop, or upload images and any file type
  • Emoji shortcodes, footnotes, callouts, definition lists
  • Instant full-text search across all notes with hashtag autocomplete
  • Three-pane layout — notes sidebar, editor, and files panel — all resizable
  • Export and import your full workspace as ZIP (notes, files, and settings)
  • Formatting toolbar with keyboard shortcuts
  • Configurable settings — theme, font size, line height, content width, trash retention
  • Sync across devices via Google Drive — direct connection, we never see your data
  • Install as a PWA — works completely offline after first visit
  • Light and dark theme with adaptive syntax highlighting

Free and open source

No premium tier, no feature gates, no ads. Built as a Progressive Web App that works across all modern browsers and devices. Source code available on GitHub.
